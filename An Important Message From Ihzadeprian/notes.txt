Website ini dibangun sebagai portofolio dari Pemrograman Mobile dengan menggunakan Laravel untuk membangun API dan Flutter untuk Aplikasi Mobile yang dibangun

Yang meliputi materi CRUD dengan menggunakan Database dan menggunakan framework Laravel 8
Bertemakan Aplikasi Mobile E-Commerce pembelian tas

Menggunakan Local Server Laragon dengan menjalankan perintah 'php artisan serve' dan mengakses http://127.0.0.1:8000/

Database berada pada file bagpacker-backend.sql jika ingin yang sudah terdapat data user
Atau bisa dengan perintah 'php artisan migrate --seed' namun isi Database kosong
